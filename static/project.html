Webserver Integrated Arduino Attendance Metric Tracker

About the Project

This project utilizes two infrared (IR) sensors placed on each side of an open doorway to detect the entry and exit of individuals from a room. When sensor 1 pings followed by sensor 2 shortly after, this indicates that someone entered a room, and when sensor 2 pings before sensor 1, this indicates that someone left the room. The Arduino microcontroller keeps track of the number of people entering and exiting the room, maintaining a running total, and updating this value within our web server. A client GET request for this data will send it to the client and to a specified email.

Key Features

  • Two IR sensors for accurate detection
  • Arduino microcontroller tracking number of people entering/exiting
  • Web server integration for updating the running total
  • GET request to retrieve the current total
  • ESMTP integration with Arduino and web server
